## 📁 Cấu trúc file

```
portfolio-website/
├── index.html          # Trang chính (Hero, About, Skills, Contact)
├── css/
│   ├── style.css         # Styles chính (layout, components, responsive)
│   └── animations.css    # Animations (fade-up, float, blink, heartbeat)
└── js/
    ├── main.js           # Navbar scroll effect, hamburger menu
    ├── typewriter.js     # Typewriter animation ở hero section
    └── scroll-animations.js  # Scroll reveal + progress bar animation
```
