# 💼 Portfolio Website

Trang portfolio cá nhân giới thiệu bản thân, kỹ năng và dự án của tôi.

## 🚀 Demo

**🔗 [Xem live tại đây](https://sinh1011.github.io/portfolio-website/)**

## ✨ Tính năng

- Trang giới thiệu cá nhân
- Hiển thị kỹ năng lập trình
- Danh sách dự án nổi bật
- Form liên hệ
- Giao diện responsive, hiện đại
- Smooth scroll animation

## 🛠️ Công nghệ

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
